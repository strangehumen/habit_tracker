# 習慣トラッカー

HTML、CSS、JavaScriptで作成した習慣管理アプリです。

## デモ

https://strangehumen.github.io/habit_tracker/

## 制作背景

毎日の習慣を継続しやすくするため、達成状況や継続日数を可視化できるアプリとして制作しました。

継続のモチベーション維持を目的とし、シンプルで見やすいデザインを意識しています。

## 主な機能

- 習慣の追加
- 習慣の編集
- 習慣の削除
- 今日の達成チェック
- 継続日数の表示
- 最長継続記録の表示
- 達成率の表示
- プログレスバーによる可視化
- 統計情報の表示
- localStorageによるデータ保存
- ダークモード
- レスポンシブデザイン

## 使用技術

- HTML
- CSS
- JavaScript
- localStorage

## 特徴

- 継続日数や達成率を可視化
- シンプルで直感的な操作性
- ブラウザを閉じてもデータを保持
- スマートフォンでも快適に利用可能
